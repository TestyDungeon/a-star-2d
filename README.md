## 💡 Overview

This project is a simple 2D game developed as part of a university project for "AI in Computer Games" course.
The challenge was to use some of the algorithms in the game. I have created it in Unity.
<div align="center">
  
[![Video](https://img.youtube.com/vi/2tc33ByLElU/0.jpg)](https://www.youtube.com/watch?v=2tc33ByLElU )
</div>

## ✨ Algorithms implemented

- A* Pathfinding  
Enemies use the A* algorithm to find the shortest path to the player.
- Depth-First Search
The map is generated using DFS.

## 🎮 How to Run

- Game executable is in /Game/ folder.
